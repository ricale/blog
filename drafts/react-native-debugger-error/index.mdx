### Cannot add node "1" because a node with that id is already in the Store.

위 에러가 발생하며 디버거가 정상 동작하지 않았다. 원인은 [react-devtools-core](https://www.npmjs.com/package/react-devtools-core) 라는, React Native 의 디펜더시의 이슈였다. 해당 디펜던시의 특정 버전을 설치해서 에러를 제거할 수 있다. package.json 파일의 `resolutions` 항목에 아래처럼 추가해주자.

```js
{
  // ...
  "resolutions": {
    // ...
    "react-devtools-core": "4.14.0"
  },
}
```

](https://github.com/facebook/react/issues/23226#issuecomment-1051006915)
